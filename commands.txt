### Docker


## Build image (DEV)
docker build -t personal-website-frontend-dev -f Dockerfile.dev .
OR
docker-compose -f docker-compose.dev.yml build

## Build image (PROD)
docker build -t personal-website-frontend-prod -f Dockerfile .
OR
docker-compose build

## Run Containers (DEV)
docker-compose -f docker-compose.dev.yml up

## Run Containers (PROD)
docker-compose up
